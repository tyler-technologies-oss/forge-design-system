## Skip to main content

<RecipeDemo href="https://stackblitz.com/edit/skip-to-main-content-link" />

Use a "skip to main content" option to meet <a href="https://www.w3.org/WAI/WCAG21/Understanding/bypass-blocks.html" target="_blank" rel="noreferrer noopener">WCAG 2 Criteria 2.4.1: Bypass Blocks</a>.
This allows users to skip repeated content blocks in order to get more direct access to the primary content of a web page.

:::info
Recommended for community apps.
:::

Meets needs for:

- Sighted keyboard users
- Users with visual impairments
